@Ayrton Mercado I have to document this, you can use this for now:

# Inside perSystem
{
  imports = [
    "${inputs.nixpkgs}/nixos/modules/misc/nixpkgs.nix"
  ];
  nixpkgs = {
    # Required for CUDA
    config.allowUnfree = true;
  };
}
